# 📝 To Do List

Aplicação de lista de tarefas desenvolvida com **React + Vite**, com foco em aprendizado de front-end moderno e boas práticas de desenvolvimento.

---

## 🚀 Deploy

👉 https://todo-list-git-main-azevedosvgs-projects.vercel.app/

---

## 📸 Preview

![Preview do projeto](./assets/preview.png)

---

## ✨ Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Filtrar por:
  - Todas
  - Concluídas
  - Pendentes

- Dark Mode com toggle animado
- Persistência com LocalStorage

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Manipulação de estado com React (`useState`)
- Uso de efeitos com `useEffect`
- Componentização
- Gerenciamento de listas e eventos
- Criação de interfaces responsivas
- Uso de variáveis CSS (Dark Mode)
- Boas práticas de organização de código
